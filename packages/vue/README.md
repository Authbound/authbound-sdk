# @authbound/vue

Vue plugin, composables, components, and styles for Authbound verification flows.

```sh
pnpm add @authbound/vue
```

```ts
import {
  AuthboundPlugin,
  VerificationWall,
  useVerification,
} from "@authbound/vue";
import "@authbound/vue/styles.css";
```

For Nuxt applications, prefer `@authbound/nuxt`.
